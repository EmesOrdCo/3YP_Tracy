# How to Visualize the Mega Diagram in Cursor

## Option 1: Markdown Preview (Recommended)

1. **Open the MEGA_DIAGRAM.md file** in Cursor
2. **Press** `Cmd+Shift+V` (Mac) or `Ctrl+Shift+V` (Windows/Linux) to open Markdown Preview
3. **If Mermaid is supported**, the diagrams will render automatically

If Mermaid doesn't render, try Option 2 or 3 below.

---

## Option 2: View Online

1. Copy the Mermaid code from any diagram section
2. Go to: https://mermaid.live/
3. Paste the code
4. View the interactive diagram

---

## Option 3: Install Mermaid Extension

1. In Cursor, open Extensions (`Cmd+Shift+X` or `Ctrl+Shift+X`)
2. Search for "Mermaid Preview" or "Markdown Preview Mermaid Support"
3. Install the extension
4. Reopen the markdown preview

---

## Option 4: View Simplified Text Diagram

See `MEGA_DIAGRAM_TEXT.md` for a text-based version that will always render.
