# Vehicle Configuration Files

This directory contains vehicle configuration files in JSON format.

## Files

- **`base_vehicle.json`** - Base/default vehicle configuration
- **`optimized_vehicle.json`** - Optimized configuration from comprehensive optimization
- **`UNITS_REFERENCE.md`** - Complete units reference for all parameters

## Units Reference

For complete parameter units, see **[UNITS_REFERENCE.md](./UNITS_REFERENCE.md)**

## Quick Units Summary

| Parameter Type | Units |
|---------------|-------|
| Mass | kg |
| Length/Distance | m (meters) |
| Moment of Inertia | kg·m² |
| Force | N (Newtons) |
| Torque | N·m |
| Voltage | V (Volts) |
| Current | A (Amperes) |
| Resistance | Ω (Ohms) |
| Power | W (Watts) |
| Angular Velocity | rad/s |
| Temperature | °C |
| Velocity | m/s |
| Rate | N·m/s |
| Efficiency/Ratio | dimensionless (0-1 or ratio) |
| Density | kg/m³ |

## Notes

- Standard JSON format is used (no inline comments supported)
- See `UNITS_REFERENCE.md` for detailed units for every parameter
- All dimensionless parameters (ratios, coefficients) have no units
